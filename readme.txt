== Description ==

Automatically redirects to your live mobile optimized website provided by Qikmo.

To use this plug in you must have a mobile site provided by Qikmo. Visit http://www.qikmo.com to view our plans and pricing or www.qikmo.com/freepreview to request a free preview of your website today.

The Qikmo Mobile Redirection plug in takes care of inserting the lines of code your website needs to properly redirect to your live mobile optimized website when viewed from a smartphone.

== Installation ==

1. Unzip the plugin
1. Upload plugin folder to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure plugin under Settings->Qikmo setup menu

Upgrading from 1.0 -> 1.2

Version 1.2 requires Handset detection API credentials to work properly. Please check plugin settings after upgrade to finish setup.


== Frequently Asked Questions ==

= What is a mobile website redirect? =
Before smartphone users can see the mobile-optimized version of a website, the site owner must insert a redirect script into the index.html file on their regular website. The Qikmo Website for WordPress plugin removes this hassle and automatically redirects your WordPress site to the mobile-optimized site when someone visits from their mobile device.

= How can I get a mobile site? =
Visit http://www.qikmo.com/freepreview to request a free preview of your existing website optimized for mobile.

= Qikmo has made my site live, how do I get started with my redirect? =
Read our support documentation at http://www.qikmo.com/re-direct-support to learn how to set up your mobile domain with a CNAME record.

= How do I activate the mobile website redirect? =
Go to WordPress Control Panel. Click "Settings" on your left-column menu, then "Qikmo Setup". Enter your mobile URL (eg: http://m.qikmo.com). Users from mobile phones are automatically redirected to the home page of your mobile site.

= Is there any special configuration that I need to use on WordPress to ensure the plugin works? =
No, you do not need to make any changes to your WordPress website in order to make the plugin work.

= Where is my mobile website hosted? =
When you optimize your website for mobile with Qikmo, we take care of the hosting and backup of your mobile site.

= How does Qikmo auto-sync my mobile website with my WordPress website? =
This is part of our secret sauce. Each time a person visits your mobile site, Qikmo goes out and fetches updated content from your desktop site. This includes text, images, video even changes to the html. All of this done in real-time with no delay for your mobile site visitors.

= Will my mobile website look good on all smartphones? =
Mobile sites built by Qikmo are designed to look great on all smartphones including iPhone, most Android phones, Windows phones and a variety of Blackberry devices. We perform extensive testing to ensure that your mobile website displays properly on new smartphones.

= Where can I get help? =
Go to Qikmo support to view Help and tutorials at http://www.qikmo.com/re-direct-support
